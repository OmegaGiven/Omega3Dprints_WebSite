Bash Commands

  • cat; concatenates files and outputs to terminal
  • more; scrolls through text
  • less; scrolls through backwards and doesnt open the file while viewing.
  • head; prints the amount of lines as imputed, ex. head -2 ...
  • tail; prints the amount of lines at the end as imputed, ex. tail -2 ...
  • mkdir; used to create a directory if not already existing.
  • cp; copies the files and directories from source path to current path.
  • mv; moves files files by copying to new directory and deleting old.
  • rm; removes a file. ex, rm readme.md
  • touch; creates or updates a file.
  • grep; searches for text in a file.
  • sort; sorts contents of a file.
  • wc; counts the number of characters and words in a file.
  • cut; cuts part of a file
  • ls; lists all the files or folders
  • cd; changes directory, ex. cd .., ex. cd /desktop
  • du; shows disk usage
  • echo; states something to the system
  • sleep; makes the command line halt for a designated time

Git Commands

  • git status; shows Status of local files(which files are changed, which files haven't been added to the repository, etc.) ex. no example needed. try it out.
  • git init; creates a new repository on the computer at the current directory. ex. no example needed. try it out to see how it works.
  • git add; adds file of the file with directory entered after command. ex. git add readme.MD
  • git config; Set and view git's configuration parameters. ex. git config --global user.name "..."
  • git commit; Permanently records changes made into the repository. ex. git commit -m "this will commit added files"
  • git remote; Associate a repository on another computer with this local repository ex. git remote add origin
  • git push; sends committed changes to another repository declared. ex. git push --upstream
  • git log; Review the history of commits. ex. no example needed.

CSS concepts

  • background-color; sets the background to a color.
  • color; sets object to a color
  • font-family; makes a set of fonts a computer can use one by one if not able to open one in particular
  • font-style; sets the font of a text.
  • text-align; sets text to be center right left etc...
  • margin; adds spacing in margin location so when moving it will not overlap that area.
  • padding; adds spacing around the text or object.
